##  Project Overview

**WellnessAI** is an AI-powered fitness and wellness web platform that combines **real-time exercise correction**, **AI guidance**, and a **professional appointment system** in one complete solution.

Currently, there is no system that provides:

* Real-time AI exercise posture correction
* Rep counting with live feedback
* AND professional physiotherapist, trainer, and dietitian appointment booking in one platform

WellnessAI solves this problem by integrating **AI fitness guidance** with a **role-based professional healthcare system**, helping users exercise safely, correctly, and with expert support.

* They do not know the correct posture

* They risk injuries due to wrong movements

Also, there is no single platform that combines:

* AI exercise correction

##  AI Exercise Module

WellnessAI currently supports **3 AI-based exercises**:

1. Bicep Curl
2. Shoulder Press
3. Front Raise

### How it works:

* User selects an exercise
* Webcam opens
* AI detects body posture
* A progress bar fills only when posture is correct
* After 10 correct reps, exercise completes
* Real-time feedback is shown to the user

This ensures **only correct repetitions are counted**, not wrong movements.
